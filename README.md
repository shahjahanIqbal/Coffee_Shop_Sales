# Coffee Shop Sales Dashboard
Interactive Excel dashboard analyzing coffee shop sales using pivot tables, slicers, and dynamic visualizations. Provides insights into yearly sales trends, top-performing countries, and high-value customers with multi-dimensional filtering.
Coffee Shop Sales Dashboard

This project is an interactive Excel-based data analytics dashboard designed to explore and visualize coffee shop sales data. It leverages pivot tables, slicers, and dynamic charts to provide actionable insights into sales performance across multiple dimensions.

**Project Overview**

The dashboard enables users to:
1. Analyze sales trends over time
2. Identify top-performing countries
3. Track top 5 customers by revenue
4. Filter data dynamically using multiple business-relevant attributes

The goal is to simulate a real-world business intelligence scenario using Excel as a lightweight analytics tool.

**Key Features**
1. Interactive Dashboard
   
Clean and structured layout for quick insights
Visual representation of sales trends and rankings
3. Pivot Table Analysis

Aggregated metrics for:
Sales by year
Sales by country
Customer-level performance
5. Dynamic Slicers

Users can filter the dashboard using:
1. Loyalty Card Status (Yes/No)
2. Coffee Roast Type (e.g., Light, Medium, Dark)
3. Product Size

These slicers allow real-time exploration of how different factors influence sales.

4. Visual Insights
Sales vs Year: Trend analysis over time
Top Countries: Geographic performance breakdown
Top 5 Customers: High-value customer identification


This dashboard helps answer questions like:

Which years had the highest sales growth?
Which countries contribute most to revenue?
Who are the most valuable customers?
How do product characteristics affect sales?

**Tools & Techniques Used**
Microsoft Excel
Pivot Tables
Pivot Charts
